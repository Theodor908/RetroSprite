# Remaining Competitive Features

## Batch C: Transform System
- Interactive scale/skew (extend existing rotation handles)

## Batch D: Tilemap Enhancement
- Live tile editing (edit one tile, all instances update)
- Isometric & hexagonal tilemap grids
- Tilemap data export (JSON/XML for game engines)

## Batch E: Import/Export
- Sprite sheet import (slice existing sheet into frames)
- APNG import

## Batch F: Input & UX
- Pressure sensitivity (tablet pen pressure → size/opacity)
- Right-click tool assignment (left = tool A, right = tool B)
- More ink modes (smear, halftone, tint, kaleidoscope)

## Batch G: Palette & Animation
- Color cycling (animated palette, Amiga-style)
- Sort palette (by hue, saturation, brightness)
- AnimPainting (auto-advance frame while drawing)

## Batch H: Advanced
- 3D layers (import .obj mesh into 2D canvas)
- Audio sync for animation
- Animated brushes (brush cycles through animation frames)
