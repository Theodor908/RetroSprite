Batch 4 Design — Architecture Overview

Three features, how they connect

Feature Summary

1. RotSprite Rotation

Pixel-art-aware rotation with live preview

  • Aseprite-style on-canvas handles
  • Draggable pivot point
  • Shift-snap to 45° increments
  • Algorithm: RotSprite (8× Scale2x → rotate → downscale)
  • Also offers Fast (nearest-neighbor) option
  • Preview-then-apply (non-destructive until Enter)

2. Layer Effects

7 non-destructive effects per layer

  • Core: Outline, Drop Shadow, Inner Shadow
  • Color: Hue/Sat Shift, Gradient Map
  • Advanced: Glow/Bloom, Pattern Overlay
  • FX button per layer in timeline
  • Effects config dialog with live preview
  • Applied during flatten, never modify source pixels
  • Saved in .retro v3 project format

3. Tilemap Layers

Tileset management + tile grid editing

  • New layer type: TilemapLayer
  • Tileset: collection of same-size tile images
  • Two modes: Draw Pixels / Draw Tiles
  • Pixel sub-modes: Auto (default) + Manual
  • Live tile updates (Pyxel Edit style)
  • Tile transforms: flip X/Y
  • Custom tile size with preset dropdown
  • Rectangular tiles only (no iso/hex)
  • Tiles panel in right sidebar

Rendering Pipeline (Updated)

// flatten_layers() — updated flow
for each layer (bottom → top):
  if TilemapLayer:
      render tile grid → pixel buffer (resolve tile refs → blit tiles)
  else:
    pixel buffer = layer.pixels

  if layer.effects:
      apply effects in order (color → pattern → glow → outline → shadow)

  composite onto frame (blend mode + opacity)

// Rotation (separate from flatten — interactive tool)
on canvas: show rotation handles when selection/layer active
  drag handle → compute RotSprite preview → overlay on canvas
  Enter → bake rotated pixels into layer
  Escape → cancel
