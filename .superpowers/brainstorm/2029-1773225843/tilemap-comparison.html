Tilemap Layers — Competitor Analysis

How Aseprite, Pixelorama, and Pyxel Edit handle tilemaps

Aseprite Model

Creating

Layer > New > New Tilemap Layer (Space+N)

Each tilemap layer owns a tileset

Tileset = collection of tile images (like a color palette)

Index 0 = empty/transparent tile

Two Modes (Space+Tab to toggle)

Draw Pixels: Edit tile content directly

  • Manual — edit without reordering tileset
  • Auto — auto-create/reuse/remove tiles
  • Stack — always create new tile variants

Draw Tiles: Place/remove tile references on grid

Strengths

  • Palette metaphor is intuitive
  • Auto mode = feels like normal drawing
  • Tile transforms (flip X/Y/diagonal)
  • 32-bit tile index with transform flags

Weaknesses

  • Confusing 3 sub-modes for pixel editing
  • Poor documentation (community complaints)
  • No auto-tiling / terrain rules

Pixelorama Model

Creating

Timeline menu > Tilemap Layer

Config dialog: choose tileset + tile size + shape

Tilesets stored in .pxo project, managed via Project Properties

Multiple tilemap layers can share a tileset

Same Two Modes (Shift+` to toggle)

Draw Pixels: Same 3 sub-modes as Aseprite (Manual/Auto/Stack)

Draw Tiles: Place from Tiles panel

  • Multi-tile selection with randomization
  • Per-tile probability weights
  • Rotation/flip via keyboard shortcuts

Strengths

  • Supports rect, isometric, and hex tiles
  • Multi-tile random placement
  • Place-only mode for level design
  • Shared tilesets across layers

Weaknesses

  • Tile size locked after creation
  • Swapping tileset risks data loss
  • Complex UI for beginners

Pyxel Edit Model

Key Differentiator: Live Tile Updates

When you edit a tile's pixels, all instances update in real-time across the tilemap. This is the killer UX feature — you see your changes propagate instantly.

Also supports: auto-import existing tileset images, identify unique tiles automatically, XML/JSON tilemap export.

Recommended Approach for RetroSprite

Cherry-pick the best from each

From Aseprite: Palette metaphor, two-mode toggle (pixel/tile), Auto sub-mode as default From Pixelorama: Tileset creation dialog with tile size config, shared tilesets, tile panel UI From Pyxel Edit: Live tile updates (edit one instance → all update), tileset auto-import Simplify: Start with rectangular tiles only (no iso/hex). Only 2 sub-modes: Auto + Manual (skip Stack). Tile flip X/Y transforms.
