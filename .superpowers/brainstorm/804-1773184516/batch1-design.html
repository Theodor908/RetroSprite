Batch 1 Design: Selection & Brushes

Three features — here's the full architecture for each

1. Freehand Lasso Tool

How it works

New class: LassoTool in src/tools.py

Interaction:

  • Click + drag to draw freeform boundary (tracked as list of (x,y) points)
  • On release → auto-close the loop (connect last point to first)
  • Fill interior using scanline algorithm → produces set((x,y))
  • Result stored in self._wand_selection (reuses existing pixel-set selection)

Visual feedback:

  • While dragging: draw the path as overlay lines on canvas
  • After release: render boundary using existing draw_wand_selection()

Files changed: tools.py, app.py (mouse handlers), canvas.py (drag preview), toolbar.py (add icon), options_bar.py

2. Selection Operations (Add / Subtract / Intersect)

Unified selection model

Key change: Unify self._selection (rect tuple) and self._wand_selection (pixel set) into a single self._selection_pixels: set[tuple[int,int]]

Rect select, wand, and lasso all produce pixel sets. This makes operations trivial:

  • Shift (Add): existing | new_pixels
  • Ctrl (Subtract): existing - new_pixels
  • Shift+Ctrl (Intersect): existing & new_pixels
  • No modifier: new_pixels (replace)

Migration: Convert rect select from storing (x0,y0,x1,y1) to producing a pixel set. Update _copy_selection, _fill_selection, _delete_selection to work with pixel sets only.

Visual: All selections rendered via draw_wand_selection() (boundary outline). Remove the separate draw_selection() rect renderer.

3. Custom Brushes (Capture from Selection)

Capture & use workflow

Capture:

  • Menu item: Edit → Capture Brush (shortcut: B or Ctrl+B)
  • Reads pixels within self._selection_pixels bounding box
  • Stores as self._custom_brush: PixelGrid + self._brush_mask: set (relative offsets from center)

Drawing:

  • When custom brush is active, PenTool/EraserTool iterate over _brush_mask offsets instead of the square size×size footprint
  • Shape-only mode: uses current foreground color + brush alpha

Reset: Switching brush size or pressing a "Reset Brush" button clears custom brush, returns to square

Files changed: app.py (capture logic, draw routing), tools.py (accept brush mask param), menu entry

Architecture Summary

Data flow
